- agent
  - 채팅에서 `@`(멘션)으로 호출가능
- instruction
  - 항상 따라야 하는 행동 지침, AI 에이전트에게 항상 자동 전달됨
- prompt
  - 채팅에서 `/`(명령 실행)으로 호출가능
  - AI 에이전트에게 실행을 지시할 행위 또는 워크 플로우
- skill
  - 채팅에서 `/`(명령 실행)으로 호출가능
  - 어떤 작업을 할때 어떤 도구를 사용하여라
  - 예를 들면 명령을 실행할때 bash를 사용하라
- plugin
  - /로 실행한다: skill, prompt, plugin workflow
  - 여러 agent와 prompt의 묶음
