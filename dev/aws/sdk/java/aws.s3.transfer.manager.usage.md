# AWSSDK S3 Transfer Manager Usage

## Docs

- 공식 API 가이드 및 가속화 가이드: [AWS S3 Transfer Manager Developer Guide](https://docs.aws.amazon.com/ko_kr/sdk-for-java/latest/developer-guide/transfer-manager.html)

- 최신 Java 레퍼런스 (Javadoc): [S3TransferManager Interface API Reference (v2)](https://docs.aws.amazon.com/java/api/latest/software/amazon/awssdk/transfer/s3/S3TransferManager.html)
