# vm.min_free_kbytes

vm.min_free_kbytes는 다음과 같은 역할을 합니다.

- 역할: 리눅스 커널이 항상 최소한으로 확보해 두어야 할 여유 메모리 공간의 크기(KiB 단위)를 정의합니다.
- 목적: 이 최소 여유 공간은 메모리 할당 요청 시 즉시 사용 가능해야 하는 커널 내부의 특수 할당(예: PF_MEMALLOC 플래그가 설정된 할당, 메모리 회수를 기다릴 수 없는 할당)을 위해 예약됩니다.
- 중요성: 이 값을 너무 낮게 설정하면 시스템 부하가 높을 때 교착 상태(deadlock)에 빠지거나 문제가 발생할 수 있으며, 너무 높게 설정하면 메모리 부족(OOM) 현상이 더 빨리 발생할 수 있습니다
